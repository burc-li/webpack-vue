
# webpack-vue

## 启动命令简介

 **eslint检测** 
```
npm run lint-ext
```

 **修复eslint检测** 
```
npm run lint-fix
```

**启动项目** 
```
npm run dev
```

 **打包** 
```
npm run build:prod
```

**分析 Webpack 开发环境打包后的模块依赖关系** 
```
npm run analyz:dev
```

**分析 Webpack 生产环境打包后的模块依赖关系** 
```
npm run analyz:prod
```
## 项目优化

1. 根据不同的使用环境合理划分 Webpack 的配置文件 
2. 使用 Eslint + husky 保证代码风格和质量 
3. 尽可能提高构建速度和压缩打包体积
